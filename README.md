# wcRay
A software path tracing program, implemented in modern C++. 

It depends on no other libraries except stb_image to read in images.

## List of Features 
- **Importance Sampling**

- **Materials**
    - Lambertian
    - Metal
    - Dielectric
    - Participating media
  
- **Texture**
    - Perlin Noise
    - Texture Mapping

- **Camera Functionalities**
    - Defocus Blur
    - Motion Blur

- **Acceleration Structure**
    - BVH
